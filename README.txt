RollingBins_DisparityandBodyMass.R - R script 
pruned_trees2024.RData - 100 consus trees extracted from Upham et al., 2019, pruned to the species analyzed 
mydat.list2024_3.RData - list of 100 copies of the dataset, containing trait distances, body mass and ecological classification. Each dataset presents its rownames matching the tip label order of each 100 pruned trees.
